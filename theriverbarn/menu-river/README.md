# เมนูอาหาร - Menu River

Web app สำหรับแสดงเมนูอาหารแบบหนังสือดิจิทัล ที่โหลดเร็วและเบา

## คุณสมบัติ

- 📖 แสดงเมนูแบบหนังสือ เปิดดูทีละหน้า
- 👆 รองรับการปัด (swipe) บนมือถือ
- 🖱️ รองรับการคลิกและลากบน desktop
- ⌨️ รองรับการกดปุ่มลูกศรบนคีย์บอร์ด
- ⚡ โหลดเร็วและเบา ใช้แค่ HTML, CSS, JavaScript
- 📱 Responsive design รองรับทุกขนาดหน้าจอ

## วิธีการใช้งาน

### การติดตั้ง

1. วางรูปภาพเมนูในโฟลเดอร์ `images/`:
   - `cover.jpg` - หน้าปก
   - `menu1.jpg`, `menu2.jpg`, `menu3.jpg`, ... - หน้าเมนู
   - `back-cover.jpg` - ท้ายปก

2. แก้ไขไฟล์ `index.html` เพื่อเพิ่มหรือลบหน้าเมนูตามต้องการ

### การควบคุม

- **มือถือ**: ปัดจากขวาไปซ้าย = หน้าถัดไป, ปัดจากซ้ายไปขวา = หน้าก่อน
- **Desktop**: คลิกที่ลูกศรซ้าย/ขวา หรือลากเมาส์
- **คีย์บอร์ด**: กดลูกศรซ้าย/ขวา

## โครงสร้างไฟล์

```
menu-river/
├── index.html      # หน้าเว็บหลัก
├── style.css       # สไตล์และ animation
├── script.js       # ตรรกะการทำงาน
├── images/         # โฟลเดอร์เก็บรูปภาพ
└── README.md       # เอกสารนี้
```

## หมายเหตุ

- รูปภาพควรมีขนาดที่เหมาะสมเพื่อให้โหลดเร็ว
- แนะนำให้ใช้รูปภาพแบบ JPEG หรือ WebP
- หน้าปกและท้ายปกจะแสดงก่อนและหลังหน้าเมนูตามลำดับ
